# JSA-vibcoding Skill 使用指南

## 一、Skill 简介

**JSA-vibcoding** 是一个专为 WPS JSA (JavaScript for Application) 宏开发设计的 AI Agent Skill。它能够帮助你：

- 🚀 **快速生成代码** - 通过自然语言描述需求，自动生成高质量 JSA 代码
- 📚 **提供完整知识库** - 涵盖单元格操作、工作表管理、数据处理等完整开发流程
- ⚡ **支持现代语法** - 支持 ES6-ES2019 语法，提供现代 JavaScript 编程体验
- 🔧 **包含实用模板** - 提供丰富的代码模板和最佳实践

### 什么是 JSA？

JSA (JavaScript for Application) 是 WPS 从 2021 版本开始支持的宏语言，基于 JavaScript：

| 特性 | 说明 |
|------|------|
| 运行时 | 内嵌 V8 引擎 |
| 语法支持 | ES6 - ES2019 |
| 兼容性 | API 与 VBA 高度兼容 |
| 索引规则 | 从 1 开始（与 VBA 一致） |

### JSA vs VBA

| 特性 | VBA | JSA |
|------|-----|-----|
| 语言基础 | Visual Basic | JavaScript |
| 变量声明 | `Dim x As Long` | `let x = 0` |
| 循环语法 | `For i = 1 To 10` | `for (let i = 1; i <= 10; i++)` |
| 数组索引 | `arr(0)` | `arr[0]` |
| 命名参数 | 支持 | 不支持 |
| 现代特性 | 较少 | 丰富（箭头函数、类、解构等） |

## 三、启用 JS 宏环境

在使用 JSA 之前，需要在 WPS 中启用 JS 宏：

### 步骤 1：设置默认开发环境

1. 打开 WPS 表格
2. 点击左上角【文件】→【选项】
3. 勾选【默认 JS 开发环境】
4. 点击确定

### 步骤 2：打开宏编辑器

- **方法一**：点击【开发工具】→【WPS宏编辑器】
- **方法二**：按快捷键 `Alt + F11`

### 步骤 3：验证环境

在宏编辑器中输入以下代码并按 `F5` 运行：

```javascript
function 测试环境() {
    alert("JSA 环境已就绪！")
    console.log("控制台输出正常")
}
```

如果弹出提示框且立即窗口显示输出，说明环境配置成功。

## 五、代码示例

### 示例 1：基础单元格操作

```javascript
function 基础操作() {
    // 读取单元格
    let value = Range("A1").Value2
    console.log("A1的值: " + value)
    
    // 写入单元格
    Range("A1").Value2 = "Hello JSA"
    
    // 批量写入（一维数组）
    Range("A1:C1").Value2 = ["姓名", "年龄", "城市"]
    
    // 批量写入（二维数组）
    let data = [
        ["张三", 25, "北京"],
        ["李四", 30, "上海"]
    ]
    Range("A2:C3").Value2 = data
}
```

### 示例 2：数据处理（推荐方式）

```javascript
function 数据处理() {
    const ws = ActiveSheet
    const lastRow = GetLastRow(ws, 1)
    
    if (lastRow < 2) {
        alert("数据不足")
        return
    }
    
    // 读取到数组（性能优化关键）
    const data = ws.Range("A2:D" + lastRow).Value2
    
    // 内存中处理
    for (let i = 0; i < data.length; i++) {
        if (data[i][1] && data[i][2]) {
            data[i][3] = data[i][1] * data[i][2]  // 金额 = 数量 × 单价
        }
    }
    
    // 一次性写回
    ws.Range("A2:D" + lastRow).Value2 = data
    
    alert("处理完成！")
}

// 辅助函数
function GetLastRow(ws, col) {
    const lastCell = ws.Columns(col).Find("*", undefined, undefined, undefined, xlByRows, xlPrevious)
    return lastCell ? lastCell.Row : 0
}
```

### 示例 4：自定义公式函数

```javascript
/**
 * 删除指定字符及后面的内容
 * 在单元格中使用: =DeleteFrom(A1, "@")
 */
function DeleteFrom(target, from) {
    let value = target.constructor.name == 'Range' 
        ? target.Value().toString() 
        : target.toString()
    
    from = from.constructor.name == 'Range' 
        ? from.Value().toString() 
        : from.toString()
    
    const index = value.indexOf(from)
    return index == -1 ? value : value.substr(0, index)
}
```

## 七、性能优化建议

### ✅ 推荐：数组批量操作

```javascript
// 快：数组批量处理
const data = ws.Range("A1:D10000").Value2
for (let i = 0; i < data.length; i++) {
    data[i][3] = data[i][1] * data[i][2]
}
ws.Range("A1:D10000").Value2 = data
```

### ❌ 避免：逐单元格操作

```javascript
// 慢：逐单元格操作
for (let i = 1; i <= 10000; i++) {
    ws.Cells(i, 4).Value2 = ws.Cells(i, 2).Value2 * ws.Cells(i, 3).Value2
}
```

### 性能检查清单

- [ ] 使用数组批量读写
- [ ] 缓存常用对象引用（如 `const ws = ActiveSheet`）
- [ ] 减少选择操作（避免 `Select()`）
- [ ] 大数据分块处理
- [ ] 使用 `const` 代替 `var`

## 九、常见问题

### Q1: 如何在 JSA 中使用可选参数？

JSA 不支持命名参数，必须按顺序传递，省略的参数使用 `undefined`：

```javascript
// VBA: AutoFilter(Field:=1, Criteria1:="苹果")
// JSA:
Range("A1").AutoFilter(1, "苹果", undefined, undefined, undefined)
```

### Q2: JSA 支持哪些 JavaScript 特性？

支持 ES6-ES2019 大部分特性：

- ✅ `let`/`const`
- ✅ 箭头函数 `() => {}`
- ✅ 模板字符串 `` `${var}` ``
- ✅ 解构赋值 `const {a, b} = obj`
- ✅ 类 `class`
- ✅ 扩展运算符 `...`
- ❌ 浏览器对象（window, document 等）

### Q3: 如何调试 JSA 代码？

1. 设置断点：在代码行号处点击
2. 使用 `console.log()` 输出到立即窗口
3. 使用 `Debug.Assert(condition)` 条件断言
4. 按 `F8` 逐语句执行

### Q4: 文件格式要求？

必须使用 `.xlsm`（启用宏的工作簿）格式，`.xlsx` 无法保存 JSA 代码。
